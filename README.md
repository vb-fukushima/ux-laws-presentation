# UXデザインの3つの法則

実務で使える心理学に基づいたUX法則のプレゼンテーションです。

## 📊 プレゼンテーション
- [スライドビューアー (index.html)](./index.html)

## 🛠 使い方

このスライドは [Marp](https://marp.app/) で作成されています。

### HTMLの書き出し
Marp CLIを使用して、MarkdownからHTMLを生成します。

```bash
# HTMLの書き出し
marp ux-laws-lt_1.md --html -o index.html

# プレビュー表示（自動更新）
marp -w ux-laws-lt_1.md --html
```

## 🎯 内容
1. **フィッツの法則（1954年）** - ボタンの配置とサイズ
2. **ミラーの法則（1956年）** - 情報の整理術
3. **ピークエンドの法則（1993年）** - 体験設計の優先順位
